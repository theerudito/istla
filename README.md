🚀 Fullstack App (Frontend + Backend)

Aplicación fullstack desarrollada con tecnologías modernas, separando claramente frontend y backend para una arquitectura escalable y mantenible.

📌 Descripción

Este proyecto consiste en una aplicación web completa que incluye:

🎨 Frontend moderno en React con TypeScript
⚙️ Backend robusto en Go
🗄️ Base de datos relacional con PostgreSQL
🔐 Autenticación basada en JWT

🖥️ Frontend

El frontend está desarrollado con un enfoque moderno, modular y tipado.

🛠️ Tecnologías usadas
⚛️ React + TypeScript
🌐 React Router → manejo de rutas SPA
📡 Axios → consumo de APIs
🧠 Zustand → manejo de estado global
🎨 Tailwind CSS → estilos rápidos y responsivos
✨ Características
Arquitectura basada en componentes
Manejo de estado global ligero y eficiente
Navegación fluida sin recarga de página
Consumo de API desacoplado
UI responsive y moderna

⚙️ Backend

El backend está construido con Go, enfocado en rendimiento, simplicidad y escalabilidad.

🛠️ Tecnologías usadas
🐹 Go (Golang) → lenguaje principal
🗄️ PostgreSQL → base de datos relacional
🔐 JWT (JSON Web Tokens) → autenticación y autorización
✨ Características
API RESTful
Manejo de autenticación con tokens
Conexión eficiente a base de datos
Código organizado por capas (handlers, services, repositories, etc.)
Alto rendimiento y bajo consumo de recursos

🔐 Autenticación

El sistema utiliza JWT para:

Login de usuarios
Protección de rutas
Validación de sesiones
